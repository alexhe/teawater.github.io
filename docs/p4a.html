在安卓平台上使用Python的方法

  • 安装QPython的APK
    • 我观察到的优点。
      • 直接可用。
      • 支持命令行模式，很多代码可以放上去就跑。
      • 支持框架。
        • Kivy
          • 它提供的代码可运行。
          • 但是我自己写了个程序放上去运行，这个程序没有正确执行。
        • Web App。
      • 自带IDE，可以在手机上直接编写代码。
      • 可以通过二维码向里面传输代码。
        • ~ http://qpython.com/#qrcode
    • 我观察的缺点。
      • 有些库可能没有。
      • 不是APK，程序的发布不便，感觉不够专业。
  • 直接用Kivy环境把程序编译成APK
    • 我观察到的优点。
      • Python代码可以编译成APK发布。
      • Kivy支持多种平台ISO，安卓，LINUX，WIN。
        • 我实际测试了同样的代码可以在UBUNTU和安卓上使用。
    • 我观察到的缺点。
      • 安卓编译环境搭建较复杂。
        • 为了解决这个问题Kivy专门提供了VBOX镜像。
        • 实际使用正确方法在Ubuntu上还是很好搭建的，后面我会介绍到。
      • 需要掌握对Kivy库的使用。
      • 安卓原生开发方法还是用JAVA，这个环境能发展到什么阶段还有待观察。
    • 安卓编译环境配置和使用。
      • 用python-for-android环境
        • ~ https://python-for-android.readthedocs.io/en/latest/quickstart/
        • 搜到的关于Kivy的中文安装就会搜到这个方法，我用过，十分麻烦且到最后编译失败。
        • 强烈不推荐。
      • 用Kivy提供的虚拟机镜像。
        • ~ https://kivy.org/docs/guide/packaging-android-vm.html
        • 我在VBOX上用了下是能跑的，但没编译APK。
        • 建议Buildozer不能用再用这个方法。
        • 具体使用方法和Buildozer一样。
      • 在LINUX下用自动脚本Buildozer
        • ~ https://kivy.org/docs/guide/packaging-android.html
        • 我在LINUX下用了这个方法，完全可用没，很方便。
        • 配置
          • git clone https://github.com/kivy/buildozer.git
            cd buildozer
            sudo python2.7 setup.py install
            • 最后的命令会自动下载需要的软件包括Android SDK和NDK。
        • 使用
          • 项目初始化。
            • 进入项目目录。
            • buildozer init
          • 项目配置。
            • 项目目录文件buildozer.spec就是配置文件，从注释可以看到默认代码文件是项目目录中的main.py。
          • 项目编码。

            • 编辑文件main.py
              import kivy
              kivy.require('1.4.1')

              from kivy.app import App
              from kivy.uix.button import Button

              class HeiBoyApp(App):
                 """docstring for MyApp"""
                 def build(self):
                     return Button(text='Hello World')

              if __name__ == '__main__':
                HeiBoyApp().run()

              注意这个代码在配置好kivy的机器都直接可用的。

          • 项目编译。
            • buildozer android debug
              • 然后apk就会出现在项目目录的bin目录了。
            • buildozer android debug deploy run
              • 增加了这两个选项还会自动部署和运行。
